Featured Projects

Gesture Identification Using Smart Glove

Developed a smart glove for sign language recognition using flex sensors and Arduino Uno. Flex sensors attached to each finger detected finger bending by measuring changes in voltage levels. Arduino Uno processed the sensor data and compared it with predefined threshold ranges to recognize specific hand gestures. The recognized gestures were converted into text and further transformed into speech using an Android text-to-speech application to assist communication.

Stress Regulation Through HRV, BPV and Cardiac Coherence Biofeedback

Developed a stress management and mental well-being system using HRV coherence biofeedback and BPV analysis. The application analyzed stress levels using ECG and PPG signals acquired through Arduino Uno and ProtoCentral MAX86150 PPG and ECG Breakout with QWIIC-v2. The system calculated coherence scores to reflect heart-brain balance and integrated guided breathing exercises with animated visualizations to support relaxation and emotional well-being. Additional features included blood pressure analysis, wellness activities, and an interactive reward-based user interface to encourage user engagement. Technologies used included Arduino IDE, VSCode, Next.js, and Openview GUI.
